# Atomic Ledger

**Immutable, append-only transaction logging.**

Atomic Ledger provides guaranteed atomic writes for financial operations. Every transaction is persisted before returning.

## Features

- **Atomic Writes** — All or nothing
- **Append-Only** — No modifications
- **Hash Chain** — Tamper detection

## Installation

```bash
uv pip install git+https://github.com/vindicta-platform/Atomic-Ledger-Py.git
```
